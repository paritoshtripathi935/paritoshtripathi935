## ▲ Release Highlights — v3.5.0

The biggest jump since `v2.0` (graduation). Faster reasoning, better incident response,
significantly improved tolerance for ambiguity. Backwards-compatible with most teams.

- **+** New backend: shipping at Parspec since `2025-04`
- **+** Two production agents on Cloudflare Workers AI — `mini-perplexity`, `mini-harvey`
- **+** Rust runtime added in experimental flag
- **~** Reverse-engineering core rewritten from scratch (Frida + JADX + mitmproxy)
- **!** Recognized as Engineer of the Quarter, Anakin · `Q3 2024`

## ⚠ Breaking Changes

- The `please-just-build-it-without-context` endpoint is **deprecated**.

- `3am-pages` are no longer supported in production builds. Use proper alerting.

## ↗ Migration Guide — How to work with `paritosh`

```http
POST /collaborate HTTP/1.1
Host: paritosh.dev
Content-Type: application/json

{
  "ask": "real problem, not a solution in disguise",
  "context": "what you've already tried",
  "constraints": ["deadline?", "scale?", "team shape?"],
  "tone": "direct"
}
```

**Response (typical):**

```json
{
  "status": 200,
  "first_reply_ms": 4200,
  "deliverable": "smallest version that proves the idea, then iterate",
  "side_effects": ["asks 'why' a lot", "writes tests"]
}
```
